1. Базовая подготовка (0–2 недели)
Что нужно:
  Основы программирования (если нет опыта):
  переменные, типы данных
  циклы, функции
  ООП: классы, объекты, наследование

Ресурсы:
  Dart Tour (официально)
  YouTube: быстрый курс по Dart

2. Dart для Flutter (1–3 недели)
Изучить:
  Null-safety
  async/await
  Future, Stream
  extensions
  mixins
  Generics
  Collections (List/Map/Set)

Мини-проекты:
  Консольный ToDo
  Парсер JSON из файла
  Асинхронный таймер

3. Основы Flutter (3–6 недель)
Изучить:
  Архитектура Flutter

Widgets:
  StatelessWidget
  StatefulWidget
  Material и Cupertino

Layout:
  Row, Column
  Stack
  Flex
  Expanded / Flexible

Навигация:
  Navigator 1.0
  routes

Мини-проекты:
  Калькулятор
  Список заметок (без сохранения)
  Простое приложение с 3–4 экранами

4. Работа с данными (1–2 месяца)
Изучить:
  HTTP (dio, http)

  Интерфейсы / репозитории
  Кэширование
  Работа с локальным хранилищем:
  SharedPreferences

  SQLite (drift)
  Мини-проекты:
  Приложение с API (погода, книги, фильмы)
